# 基于姿态检测的人体坐姿健康守护软件

本项目正在整理开源代码中，暂无详细内容。敬请期待！

利用姿态检测技术，帮助办公族保持良好的坐姿习惯，改善工作和生活中长时间坐姿所带来的健康问题，养成良好的坐姿习惯。
采用深度学习技术，结合姿态检测算法，实现对用户坐姿的实时监测和分析。建立一个高效、准确的姿态检测模型。
允许自定义检测类别，检测模型，提醒方式，语音包，打造您自己的健康生活小助手。

## 技术栈

- **语言**：Python
- **深度学习框架**：Tensorflow-keras
- **界面**：PyQt6
- **主要技术**：esp32 机器视觉 卷积神经网络
